Portfolio Project 1

Technologies Used:
  • Laravel 8
  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap 5
Project Details:

During my one-month internship at BPJS Kesehatan, I worked on a project to create a system for tracking participants who are deceased or inactive, using Laravel 8. The goal of the project was to simplify the management of BPJS Kesehatan cardholder data, ensuring the system is efficient and accurate. The system is fully responsive, meaning it works well on both desktop and mobile devices, offering a great user experience.
